Validators

Nym Validators provide privacy-enhanced credentials based on the testimony of a set of decentralized, blockchain-based issuing authorities.

Nym Validators use a signature scheme called Coconut. This allows users of the system to generate credentials through decentralised authorities, then use them with service providers.
